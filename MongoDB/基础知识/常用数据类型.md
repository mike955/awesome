### mongodb 数据类型

| 数据             | 类型                 |
| ---------------- | -------------------- |
| string:          | 字符串               |
| inter:           | 32/64 位整型         |
| boolean:         | true/false           |
| double:          | 浮点类型             |
| min/max:         | keys                 |
| arrays:          | 数组                 |
| timestamp:       | 时间戳               |
| object:          | 文档类型             |
| null:            | 空值                 |
| symbol:          | 定义一个唯一的字符串 |
| date:            | 时间                 |
| object ID:       | 文档 id              |
| binary data:     | 二进制数据           |
| code:            | javascript 代码      |
| regular express: | 正则表达式           |
