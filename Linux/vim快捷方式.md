# vim 常用快捷方式

## 命令模式（包含底线命令模式）快捷键
当前模式下输入的字符会被识别炜命令，而非字符
```vim
i               # 进入输入模式
dd              # 删除当前行
ndd             # 删除光标下面 n 行
x               # 删除当前光标所在处的字符
X               # 删除当前光标前一个字符
nx              # 向后删除 n 个字符
yy              # 复制光标所在那一行
nyy             # 复制光标下面 n 行
y0              # 复制光标所在字符到行首的所有字符
y$              # 复制光标所载字符到行尾的所有字符
y[n]l           # 复制光标右边 n 个字符
y[n]h           # 复制光标坐标 n 个字符
p（小写）        # 将复制的数据粘贴在光标之后
P（大写）        # 将复制的数据粘贴在光标之前
:               # 切换到底线命令模式
hjkl            # 光标(左下上右)
（ctrl + f）Page Down       # 向下翻页
（ctrl + b）Page Up         # 向上翻页
ctrl + d        # 向下翻半页
ctrl + u        # 向下翻半页
+               # 光标移动到下一行
-               # 光标移动到上一行
n<space>        # n 表示数字，输入数字后按下空格键，光标向右移动 n 个字符，输入数字时，数字是不可见的
n<Entry>        # 光标向下移动 n 行
$               # 移动到这一行最后一个字符
0(^)            # 移动到这一行的首字母 
nG              # n 表示数字，光标移动到第 n 行
gg              # 移动到文档首行
/word           # 向光标后面寻找名称为`word`的字符
?word           # 向光标前面寻找名称为`word`的字符
n               # n 为英文按键。重复前一个搜索动作
N               # N 为英文按键，向反方向重复前一个搜索动作 
u               # 撤销上一次操作
[ctrl] + r      # 前进上一个操作（撤销的反向操作）

:set nu         # 显示行号
:set nonu       # 取消行号
:2              # 光标跳到第二行首字母
:$              # 光标跳到末尾行首字母
:w              # 保存修改
:q              # 退出
:x              # 保存并退出
:q!             # 强制退出，有未保存修改时放弃修改
:n1,n2s/word1/word2/g   # n1,n2 为数字，在 n1 与 n2 行之间寻找 word1 这个字符，并将所有的 word1 字符替换为 word2 字符
:1,$s/word1/word2/g     # 将当前文本中所有的 word1 字符替换为 word2 字符
:1.$s/word1/word2/gc    # 将当前文本中所有的 word1 字符替换为 word2 字符，并在替换前询问用户是否确认替换
```

## 输入模式快捷键
当前模式下输入的字符会被识别为字符
```vim
esc             # 退出输入模式，进入命令模式
```

## 底线命令模式
在命令模式下输入`:`（英文冒号）进入底线命令模式
```vim
:set nu         # 显示行号
:2              # 光标跳到第二行首字母
:$              # 光标跳到末尾行首字母
:w              # 保存修改
:q              # 退出
:q!             # 强制退出，有未保存修改时放弃修改
```
